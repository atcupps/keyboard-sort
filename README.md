# Keyboard Sortcuts

A SvelteKit application for visualizing sorting algorithms, designed for keyboard-driven interactions.

## Current Status
- Basic visualization with 10 randomized bars.
- Tallest bar is 10x the height of the base height.
- Simple "Randomize" button for generating new sets.

## Getting Started
1. Install dependencies: `npm install`
2. Run in development: `npm run dev`
3. Build for production: `npm run build`
